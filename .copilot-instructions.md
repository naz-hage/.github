# Workspace Master Instructions (Source of Truth)

**Portable. Generic. Works for all team members.**

This is the **authoritative source** for all Copilot instructions and skills in this workspace. It automatically applies to all workspace folders, regardless of installation location or folder structure.

## Authority & Scope

- **Owner**: Located in the workspace configuration folder (`.github`)
- **Skills**: `.skills/` folder with all skill definitions and templates
- **Applies to**: ALL workspace folders automatically via wildcard pattern matching
- **Updates**: Central point of maintenance—no duplication across folders
- **Portability**: No hardcoded paths; works when cloned to any location
- **Access**: Automatically available when `.code-workspace` is opened

See [WORKSPACE-AUTHORITY.md](./WORKSPACE-AUTHORITY.md) for detailed authority structure.
